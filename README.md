# Сортировка вставками
## Описание
На вход алгоритма подаётся последовательность n чисел:a1,a2,...,aN. Сортируемые числа также называют ключами. Входная последовательность 
на практике представляется в виде массива с n элементами. На выходе алгоритм должен вернуть перестановку исходной последовательности 
a'1,a'2,...,a'N, чтобы выполнялось следующее соотношение a'1 <= a'2 <= ...<= a'N.
В начальный момент отсортированная последовательность пуста. На каждом шаге алгоритма выбирается один из элементов входных
данных и помещается на нужную позицию в уже отсортированной последовательности до тех пор, пока набор входных данных не будет исчерпан. 
В любой момент времени в отсортированной последовательности элементы удовлетворяют требованиям к выходным данным алгоритма.
